### Assumed Background

This course assumes familiarity with:

- linked structures vs array-based structures,
- basic classes and methods.

### Course Topics

- Foundations: what spatial data is
- Spatial representations (points, lines, polygons; raster vs vector)
- Coordinates and reference systems
- Distance, space, and measurement
- Extent, scale, and resolution
- Spatial data formats (GeoJSON, CSV + coordinates, shapefiles/raster overview)
- Working with real spatial data
- Spatial relationships and queries
- Spatial analysis concepts (buffers, clipping, overlays, joins)
- Visualization and communication
- Interactive and programmatic mapping
- Scaling and performance awareness
- Time/change concepts and applied spatial thinking
