## 13 — Installing and Using the WDO Library

### Goal

Install and use the reusable helper code stored in `src/wdo`.

### Students will practice

- understanding the `src/` project layout and why it exists
- reading `pyproject.toml` and knowing what each section does
- running `pip install -e .` for an editable install
- importing helper functions from `wdo.geo` and `wdo.maps`
- replacing inline notebook code with library calls
- modifying `wdo` and reloading in-notebook

### Why this matters

At this point students have seen enough repeated geometry code that packaging helper functions now feels useful instead of mysterious.
This lesson closes the loop from Module 12: the refactored library is now installed and used end-to-end.
